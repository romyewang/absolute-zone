# 超级玛丽任务管理系统
一个基于网页的任务和精力资源管理工具，采用超级玛丽像素风格设计。

## 功能特性

- 🎮 **超级玛丽风格界面**
- 📋 **三大模块管理**：工作、学习、个人成长
- ✅ **任务管理**：创建、完成、删除任务
- ⭐ **积分系统**：完成任务获得积分
- 📎 **文档管理**：文件上传、下载
- 💾 **本地存储**：数据自动保存
## 使用方法

1. 选择模块（工作/学习/个人成长）
2. 输入任务内容并添加
3. 完成任务获取积分
4. 上传相关文档文件
5. 查看积分进度

## 技术栈

- HTML5
- CSS3（像素风格设计）
- JavaScript（ES6+）
- LocalStorage（数据存储）

## 部署

直接打开 `index.html` 即可使用，无需服务器支持。
